# [과제] 1주차 - 아토믹 컴포넌트

## 목표

- 아토믹 컴포넌트 구현 : Figma 컴포넌트 → React 컴포넌트 구현
- 바닐라 프로젝트 결과물에서 구현할 컴포넌트를 1개 선정합니다.
- Figma를 사용해 선정한 컴포넌트 정의 및 변형을 설계합니다.
- 디자인된 컴포넌트를 React 컴포넌트로 구현합니다.

## 수행조건

- 모두가 접근 가능
- 의미있는 구조 설계
- 체계적인 표현 설계
